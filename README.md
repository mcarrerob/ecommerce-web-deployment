# Learning App E-commerce (Ubuntu / LAMP)

Este proyecto es una adaptación para Linux/Ubuntu de la app de ejemplo de e-commerce de KodeKloud.

## Tecnologías

- Linux (Ubuntu)
- Apache2
- MariaDB
- PHP
- HTML/CSS/JS

## Créditos

Proyecto original: `kodekloudhub/learning-app-ecommerce`.
